# 同行 · 产品文档

## 产品概览
同行是一款帮助用户直观感知时间流逝的日常效率工具。应用以一天为单位，将 24 小时均分为 144 个 10 分钟小格，采用简洁柔和的配色呈现今日进度，让用户在打开 App 的瞬间即可了解「今天已经过去了多少」与「还剩下多少」。

## 核心功能
- **实时进度网格**：主页展示 12×12 共 144 个方块，对应当天的 144 个 10 分钟时间段。方块颜色随时间变化，已过去的时间块（Passed）会被高亮，未到的时间块（Left）保持淡色。
- **即时统计**：顶部醒目显示今日已过去的方块数量与剩余数量，并同步标注对应的累计分钟数，帮助用户快速把握时间成本。
- **温和视觉反馈**：界面背景采用米色基调，网格方块以绿色渐变体现温度感，既突出关键信息，又维持整体视觉舒适度。

## 目标用户
- 希望提升时间意识的职场人士与学生
- 习惯以番茄钟或 10 分钟为粒度规划日程的效率控
- 喜欢用数据化方式记录与提醒自我的个人成长用户

## 使用场景
- 开启工作或学习前，查看当前时间所处位置，明确剩余可用时间。
- 全天多次打开 App，及时感知时间流速，调整计划或休息节奏。
- 睡前回顾：回望今日 144 格中高亮比例，反思时间利用率。

## 设计亮点
- **极简操作**：打开即见核心信息，无需滑动或点击即可掌握当下。
- **日历感增强**：网格排布类似桌面日历，帮助用户更快形成记忆并建立使用习惯。
- **情绪友好**：柔和的色彩与圆角方块降低时间压力，鼓励用户以更从容的心态面对日常。

## 未来规划
- 支持自定义颜色主题，满足不同用户的视觉偏好。
- 引入简单的目标追踪与提醒功能，将时间认知与目标管理结合。
- 提供统计视图，展示过去数日的进度比例，帮助用户持续优化时间管理。

## 启动说明
当前为 Expo 应用，开发者可通过以下命令启动项目：

```bash
npm install
npx expo start
```
